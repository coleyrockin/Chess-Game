# Code of Conduct

## Our standards

We expect contributors to be respectful, direct, and constructive.

Examples of expected behavior:
- Giving and receiving actionable technical feedback
- Respecting different experience levels and viewpoints
- Focusing discussion on code and project goals

Unacceptable behavior includes harassment, personal attacks, or discriminatory language.

## Enforcement

Project maintainers may remove comments, issues, or pull requests that violate this code of conduct.

## Scope

This policy applies to this repository's issues, pull requests, and discussions.
