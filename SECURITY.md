# Security Policy

## Supported versions

This project is under active development. Security fixes are applied to the latest `main` branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these private channels:
- Open a private security advisory in the repository Security tab.
- If advisories are unavailable, open a private maintainer contact through GitHub profile messaging.

Include:
- A clear description of the issue
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation (if known)

You will receive acknowledgement as soon as possible, and validated issues will be fixed in a prioritized patch.
